# تطبيق خليل القرآن - السياسات والخصوصية 📜

هذا المستودع مخصص لاستضافة الوثائق القانونية وسياسات الخصوصية الخاصة بـ **تطبيق خليل القرآن**، وهو نظام متكامل لإدارة مراكز تعليم القرآن الكريم ومتابعة الطلاب، تم تطويره لصالح **مركز خليل الرحمن**.

## 📖 حول تطبيق خليل
تطبيق خليل القرآن هو منصة تقنية متخصصة تهدف إلى تسهيل العملية التعليمية داخل مراكز القرآن الكريم، ويوفر أدوات متقدمة لكل من:
* **المعلمين:** لمتابعة حضور الطلاب وتسميع الأجزاء اليومي.
* **الإدارة:** لمراقبة أداء المركز والحصول على تقارير إحصائية دقيقة.
* **أمن البيانات:** ضمان الحفاظ على إرث القرآن الكريم باستخدام أحدث الوسائل التقنية المؤمنة.

## 🔒 التزامنا بالخصوصية
بما أن المنصة تخدم مؤسسات تعليمية، فإن خصوصية البيانات هي أولويتنا القصوى:
* **تشفير البيانات:** نستخدم بروتوكولات تشفير قياسية (HTTPS) لحماية البيانات أثناء انتقالها.
* **عدم مشاركة البيانات:** بيانات الطلاب والمعلمين سرية تماماً ولا يتم مشاركتها أو بيعها لأي جهة خارجية.
* **الشفافية:** يمتلك المستخدمون التحكم الكامل في بياناتهم داخل التطبيق.
